# DiVPN
Full-stack VPN application built for iOS.

The client is written in SwiftUI using the MVVM architecture pattern, with a focus on clean state management, modularity, and native performance. The project also includes custom Metal shaders for GPU-accelerated visual effects.

The backend is built with ASP.NET and PostgreSQL, providing authentication, subscription management, VPN configuration delivery, and server-side infrastructure APIs.

# Tech Stack

## iOS Client
SwiftUI
MVVM
Metal Shaders
NetworkExtension
Combine

## Backend
ASP.NET
PostgreSQL
REST API
